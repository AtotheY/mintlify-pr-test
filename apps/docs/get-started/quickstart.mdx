## Create a New Project

Create a new Spinai project using our CLI:

<CodeGroup>

```bash npm
npx create-spinai-app my-agent
```

```bash yarn
yarn create spinai-app my-agent
```

</CodeGroup>

This will create a new directory with a fully configured Spinai project including:

- Example actions for ticket management
- OpenAI orchestrator configuration
- TypeScript configuration
- Environment setup

```bash
my-agent/
├── actions/
│   ├── createTicket.ts
│   └── getCustomerInfo.ts
├── orchestrator/
│   └── openai.ts
├── .env.example
├── spinai.config.ts
└── tsconfig.json
```

## Create Your Agent

```typescript src/index.ts
import { createAgent } from "spinai";
import { OpenAI } from "openai";
import { createTicket } from "./actions/createTicket";
import { getCustomerInfo } from "./actions/getCustomerInfo";

const openai = new OpenAI();

const supportAgent = createAgent({
  instructions: "A support agent that helps manage customer tickets.",
  actions: [createTicket, getCustomerInfo],
  responseFormat: "json",
  model: openai.chat.completions.create,
});

export default supportAgent;
```

## Start Development

1. Install dependencies:

```bash
cd my-agent
npm install
```

2. Configure your environment:

```bash
cp .env.example .env
# Add your OpenAI API key to .env
```

3. Start the development server:

```bash
npm run dev
```

Your AI agent is now running at `http://localhost:8080`!

## Test Your Agent

Send a POST request to test your agent:

```bash
curl -X POST http://localhost:8080/api/run \
  -H "Content-Type: application/json" \
  -d '{"input": "Create a new ticket for customer support"}'
```
