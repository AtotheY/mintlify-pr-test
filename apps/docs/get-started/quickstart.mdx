## Installation

Get started with Spinai by installing it in your project:

<CodeGroup>

```bash npm
npm install spinai
```

```bash yarn
yarn add spinai
```

</CodeGroup>

## Basic Setup

### 1. Create Your Project Structure

Create the following directory structure for your AI agent:

```bash
my-ai-agent/
├── actions/
│   └── example.ts
├── orchestrator/
│   └── index.ts
└── spinai.config.ts
```

### 2. Configure Your Orchestrator

Create an OpenAI orchestrator in `orchestrator/index.ts`:

```typescript orchestrator/index.ts
import { createOpenAIOrchestrator } from "spinai";

export default createOpenAIOrchestrator({
  apiKey: process.env.OPENAI_API_KEY,
  systemPrompt: `You are an AI assistant that helps users by coordinating actions.
  Analyze user input and decide which actions to run.
  Respond with clear reasoning about your decisions.`,
});
```

### 3. Create an Action

Create your first action in `actions/example.ts`:

```typescript actions/example.ts
import type { ActionModule } from "spinai";

export const config = {
  id: "example",
  metadata: {
    description: "A simple example action",
  },
};

export async function run(context) {
  const { input } = context.request;
  return `Processed: ${input}`;
}
```

### 4. Configure Spinai

Set up your `spinai.config.ts`:

```typescript spinai.config.ts
import { SpinaiConfig } from "spinai";

export default {
  actionDirectoryPath: "./actions",
  orchestratorDirectoryPath: "./orchestrator",
} satisfies SpinaiConfig;
```

### 5. Start the Server

Create an entry point file (`index.ts`):

```typescript index.ts
import { runAgentServer } from "spinai";

runAgentServer(8080);
```

Run your agent:

```bash
ts-node index.ts
```

Your AI agent is now running at `http://localhost:8080`!

## Test Your Agent

Send a POST request to test your agent:

```bash
curl -X POST http://localhost:8080/api/run \
  -H "Content-Type: application/json" \
  -d '{"input": "Hello, Spinai!"}'
```
