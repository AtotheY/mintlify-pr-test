## Starting the Server

Create an entry point file (`index.ts`):

```typescript index.ts
import { runAgentServer } from "spinai";

runAgentServer(3000);
```

Run your agent:

```bash
ts-node index.ts
```

## Making Requests

Send requests to your agent:

```bash
curl -X POST http://localhost:3000/api/run \
  -H "Content-Type: application/json" \
  -d '{
    "input": "Handle this support ticket",
    "metadata": {
      "ticketId": "12345"
    }
  }'
```

## Monitoring

Monitor your agent's performance using Spinai's built-in logging:

```typescript
import { log } from "spinai";

log("Processing ticket", {
  level: "info",
  data: { ticketId: "12345" },
});
```

## Production Deployment

For production deployment:

1. Set up error monitoring
2. Configure rate limiting
3. Add authentication
4. Set up logging
5. Configure environment variables

Example production configuration:

```typescript
import { runAgentServer } from "spinai";
import rateLimit from "express-rate-limit";
import auth from "./middleware/auth";

const app = await createServer(config);

// Add rate limiting
app.use(
  rateLimit({
    windowMs: 15 * 60 * 1000, // 15 minutes
    max: 100, // limit each IP to 100 requests per windowMs
  })
);

// Add authentication
app.use(auth);

app.listen(process.env.PORT || 3000);
```
