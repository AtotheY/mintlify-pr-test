## Overview

This example shows how to build a Zendesk support agent that can:

- Get customer information
- Check ticket status
- Create and update tickets
- Respond to common inquiries

## Quick Start

```bash
npm create spinai-app@latest zendesk-agent
cd zendesk-agent
```

Install Zendesk dependencies:

```bash
npm install @zendesk/api
```

## Project Structure

```bash
zendesk-agent/
├── src/
│   ├── actions/
│   │   ├── getCustomerInfo.ts
│   │   ├── getTicketStatus.ts
│   │   └── createTicket.ts
│   └── index.ts
├── .env
└── package.json
```
